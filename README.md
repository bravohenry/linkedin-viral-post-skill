## Advanced: The Algorithm Game

**First 60 minutes matter most.**

LinkedIn algorithm rewards:
- Fast engagement (comments in first hour)
- Meaningful interactions (not just likes)
- Dwell time (people reading, not skimming)

**Your checklist**:
1. Post Tue-Thu, 8-10am local time
2. Respond to EVERY comment in first hour
3. Ask questions that spark discussion
4. Track metrics: impressions, engagement rate, comments
